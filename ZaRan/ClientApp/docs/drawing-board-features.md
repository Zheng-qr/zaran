# 画板控件功能说明

## 概述

基于 Fabric.js 开发的功能完整的画板控件，支持多种绘图工具和操作功能。

## 主要功能

### 1. 选择工具 🔍
- **功能**: 选择、移动、旋转和调整对象大小
- **操作**: 
  - 单击选择单个对象
  - 拖拽选择多个对象
  - 右键显示上下文菜单（复制/删除）
  - 拖拽控制点调整大小和旋转

### 2. 画笔工具 🖌️
- **功能**: 自由绘画
- **设置**:
  - 粗细: 1-50px 可调
  - 颜色: 任意颜色选择
  - 晕染效果: 可选择晕染大小
- **特性**: 支持压感和平滑绘制

### 3. 橡皮擦工具 🧽
- **功能**: 真正的擦除功能（非白色覆盖）
- **设置**:
  - 大小: 5-50px 可调
- **特性**: 能够擦除任何绘制内容，包括图片和形状

### 4. 形状工具 🔷
- **支持形状**:
  - 长方形
  - 圆形
  - 三角形
  - 五角星
- **设置**:
  - 填充颜色
  - 边框颜色
  - 边框粗细
- **操作**: 拖拽绘制，实时预览

### 5. 文字工具 📝
- **功能**: 插入可编辑文字
- **设置**:
  - 颜色选择
  - 字体大小: 12-72px
  - 字体类型
- **特性**: 双击进入编辑模式

### 6. 图片工具 📷
- **功能**: 插入图片
- **支持格式**: JPG, PNG, GIF, WebP
- **操作**:
  - 上传本地图片
  - 自动缩放适配
  - 可旋转和调整大小

### 7. 画布操作
- **缩放功能**:
  - 放大: 最大300%
  - 缩小: 最小10%
  - 重置: 回到100%
- **背景设置**: 任意颜色背景
- **清空画布**: 一键清除所有内容

### 8. 历史记录
- **撤销/重做**: 支持最多50步操作历史
- **自动保存**: 每次操作自动记录状态

### 9. 导出导入功能
- **导出图片**:
  - 格式: PNG, JPG, SVG
  - 高质量输出
- **导出数据**: JSON格式保存画布状态
- **导入数据**: 加载之前保存的画布

### 10. 右键菜单
- **复制**: 复制选中对象
- **删除**: 删除选中对象
- **取消**: 关闭菜单

## 技术特性

### 性能优化
- 懒加载 Fabric.js 库
- 事件防抖处理
- 内存管理优化

### 响应式设计
- 移动端适配
- 触摸操作支持
- 自适应布局

### 浏览器兼容性
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 使用方法

### 基本使用
```vue
<template>
  <FinalDrawingBoard />
</template>

<script setup>
import FinalDrawingBoard from '~/components/FinalDrawingBoard.vue'
</script>
```

### 高级使用
```vue
<template>
  <FinalDrawingBoard ref="drawingBoard" />
  <button @click="exportImage">导出图片</button>
</template>

<script setup>
import FinalDrawingBoard from '~/components/FinalDrawingBoard.vue'

const drawingBoard = ref()

const exportImage = () => {
  drawingBoard.value?.exportImage()
}
</script>
```

## API 接口

### 暴露方法
- `exportImage()`: 导出为图片
- `exportData()`: 导出画布数据
- `clearCanvas()`: 清空画布
- `getCanvasData()`: 获取画布数据
- `setCanvasData(data)`: 设置画布数据

### 事件监听
- `canvas:ready`: 画布初始化完成
- `object:selected`: 对象被选中
- `drawing:complete`: 绘制完成

## 注意事项

1. **性能考虑**: 大量对象时可能影响性能
2. **内存管理**: 及时清理不需要的历史记录
3. **文件大小**: 导出的图片大小取决于画布尺寸和内容复杂度
4. **浏览器限制**: 某些浏览器对文件下载有限制
